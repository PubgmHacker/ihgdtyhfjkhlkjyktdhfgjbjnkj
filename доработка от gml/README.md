# Доработка от GML

Эта папка содержит материалы и изменения по плану доработки проекта SOULDAWN от GML 5.2.

## Фазы работы

- **Фаза 1** — Фиксы авторизации (приоритет)
- **Фаза 2** — Фундамент дизайн-системы
- **Фаза 3** — Ключевые компоненты
- **Фаза 4** — Страницы
- **Фаза 5** — Контент бренда
- **Фаза 6** — Миниапп
- **Фаза 7** — Сборка, тест, пуш
